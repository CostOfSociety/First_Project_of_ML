Исследование клиентов компании "В один клик!"

Цель исследования - увеличить покупательскую активность клиентов "В один клик"

Стоит ответить на следующие вопросы:
1. Какие клиенты снизят покупательную активность в след. 3 месяца  
2. Какие клиенты относятся к какому-то сегменту

**Данные по поведению покупательной активности получаем из файла market_file.csv**

**Данные о том сколько клиент потратил получаем из файла market_money.csv** 

**Данные о времени (в минутах), которое покупатель провёл на сайте в течение периода получаем из файла market_time.** 

**Данные о о среднемесячной прибыли покупателя за последние 3 месяца получаем из файла money.csv.**

О качестве данных ничего не известно. Поэтому перед проверкой гипотез понадобится обзор данных. Затем, на этапе предобработки буду искать возможности исправить самые критичные ошибки данных.

Таким образом, исследование пройдёт так:

1. Загрузка данных.
2. Предобработка данных.
3. Проверка гипотез.
4. Объединение таблиц
5. Корреляционный анализ
6. Поиск наилучший модели
7. Факторный анализ
8. Сегментация
9. Выводы


Были использованы следующие модели: 

DecisionTreeClassifier
KNeighborsClassifier
LogisticRegression
SVC


Лучшая модель оказалась - LogisticRegression с параметрами C=6, penalty='l1', solver='liblinear', ROC_AUC составляет 0,91
